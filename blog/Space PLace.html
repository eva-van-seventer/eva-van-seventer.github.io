Content:

The content contains all the images and text. This can be thought of as the core of all the shells.

How do you control space in code??

There are 4 different layers of space areas around elements which are easily adjusted. This blog describes the difference of the layers and how to adjust them!

Padding:

Padding is the space between the content and the border, basically like the closest shell. The padding creates the space inside the element, and wraps around the content. The padding can never go outside of the border. It usually is transparent. You can adjust this by using the function 'padding'. You can code this in shorthand by labeling the lengths of padding going clockwise (top, right, bottom,left)

Border:

The border separates the margin from the padding, it usually is opaque. You can adjust this by using the function border and labeling first the width of the border, then the type of border, and finally the colour.

Margin:

The margin is the space around the object, it's basically the outer shell which wraps around everything inside it. You can adjust this, similar to how you adjust padding by labeling the length of margin going clockwise.
